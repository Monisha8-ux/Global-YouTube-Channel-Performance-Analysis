**Global YouTube Channel Performance Analysis**
**Project Overview**

This project focuses on analyzing global YouTube channel statistics to understand the factors that influence channel popularity, subscriber growth, and video engagement. Using Python-based data analysis techniques, the dataset is explored to identify patterns across different categories, countries, and channel activities. The goal is to extract meaningful insights that explain how successful YouTube channels perform.

**Problem Statement**

The objective of this project is to analyze global YouTube statistics to identify the key factors affecting channel popularity, subscriber count, and video views. By performing exploratory data analysis on the dataset, the project aims to discover trends related to channel categories, upload frequency, and geographical distribution. These insights help understand what drives engagement and success on YouTube.

**Dataset**

The dataset used in this project contains statistics of YouTube channels worldwide.

Typical attributes include:

Channel name

Category

Country

Subscribers

Video views

Total uploads

Earnings and ranking information

This dataset helps analyze global YouTube trends and channel performance.

**Key Analysis Performed**

The project answers several analytical questions such as:

Which categories have the highest average video views?

Which countries have the most successful YouTube channels?

Does the number of uploads affect total views?

Which channels have the highest subscribers?

What trends exist between channel activity and popularity?

**Sample Visualizations**

The project includes multiple visualizations such as:

Category-wise average views

Country-wise channel distribution

Uploads vs video views relationship

Correlation heatmap of numerical features

These visualizations help in understanding patterns and trends in YouTube data.

**Key Insights**

Some insights discovered during analysis include:

Certain content categories generate significantly higher average views.

Countries like the United States and India dominate the YouTube creator ecosystem.

A higher number of uploads does not always guarantee more views.

Channel category and content type play an important role in audience engagement.
